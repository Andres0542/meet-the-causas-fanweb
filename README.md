# Fanweb de Meet The Causas

Esta pagina web fue hecha para promocionar a un grupo de cosplayers peruanos mejor conocidos como los Meet The Causas. Esta no es una pagina oficial, sin embargo, tiene el permiso del grupo para que el codigo pueda estar publicado en GitHub.


# Construyendo el ambiente de desarrollo
Para instalar las dependencias necesarias para ejecutar el proyecto, solo se debe realizar el siguiente comando desde el directorio del repositorio:


```sh
npm install
```

El estilo esta contruido en base a CSS. En un futuro espero poder editar este proyecto para adaptar el codigo a SCSS  y sea mas sencillo de comprender.
